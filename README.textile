h1. Try ImageMagick

A browser-based command line for the "ImageMagick":http://www.imagemagick.org image editing software suite.

h2. Requirements

This application requires:

* Ruby version 2.0.0
* Rails version 4.0.2

h2. Database

This application uses PostgreSQL with ActiveRecord.
